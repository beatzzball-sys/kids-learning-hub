# Kids Learning Hub MVP

This is the first simple version of the kids learning game.

Included:

- React + TypeScript + Vite setup
- ABC Adventure game
- Panda Buddy mascot
- Text-to-speech pronunciation
- Stars reward system
- Progress saved in browser localStorage
- Phaser installed for future mini-games

## How to start in VS Code

1. Unzip this folder.
2. Open the folder in VS Code.
3. Open Terminal in VS Code.
4. Run:

```bash
npm install
npm run dev
```

5. Open the local URL shown in the terminal, usually:

```text
http://localhost:5173/
```
